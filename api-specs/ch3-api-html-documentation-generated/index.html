post /api/orders
send client's order (apiOrdersPost)
By sending the order details and credit car number, you can retrieve the information about the success or failure of the order

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

Body Parameter — Order request object

Return type
